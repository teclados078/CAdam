CAdam: Context-Adaptive Moment Estimation for 3D Gaussian Densification in Generative Distillation

SeungJeh Chung, Geonho Park, Misong Kim, HyeongYeop Kang
Kyung Hee University, Korea University
SIGGRAPH 2026 Conference Papers
arXiv Code

Abstract

Adaptive densification is central to 3D Gaussian Splatting, but the same mechanism becomes unreliable in optimization-based generative distillation. Because generative guidance is stochastic, standard magnitude-based densification tends to mix short-lived noise with persistent geometric cues, leading to redundant Gaussian primitives or under-fitting.

CAdam resolves this densification dilemma by reframing density control as a signal verification problem. It uses context-adaptive moment estimation to accumulate consistent gradient trends while suppressing stochastic fluctuations, then combines quantile-based context awareness with an intrinsic signal-to-noise gating mechanism. Across SDS, ISM, and VFDS objectives with strong generative 3DGS backbones, CAdam substantially reduces Gaussian count while maintaining comparable perceptual quality.
